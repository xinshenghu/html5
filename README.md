# 🍃 HTML5 项目合集

一个包含多个有趣的 HTML5 应用和游戏的项目集合，涵盖工具、娱乐、内容、生活四大领域。

## 📁 项目结构

```
html5/
├── index.htm                    # 主入口（树状导航）
├── README.md                   # 项目说明文档
├── 工具类/                      # 实用工具
│   └── tools/
│       ├── luping.html          # 屏幕录制工具
│       └── tianqi/             # 天气预报应用
├── 娱乐类/                      # 游戏娱乐
│   └── games/
│       ├── fanzhuanpintu/       # 翻转拼图游戏
│       ├── sliding-puzzle/      # 火影忍者滑块拼图
│       ├── snake/               # 贪吃蛇
│       ├── minesweeper/         # 扫雷
│       ├── match3/              # 消消乐
│       ├── sudoku/              # 数独
│       ├── frog/                # 青蛙过河
│       └── smallgame/           # 微信小程序游戏合集
├── 内容类/                      # 知识内容
│   ├── 诗词/                    # 诗词展示
│   ├── 小说/                    # 小说阅读
│   ├── 西游记/                  # 西游记内容
│   ├── 英语/                    # 英语学习
│   ├── 论语/                    # 论语展示
│   └── 内容/长安模板/           # 网页模板
└── 生活类/                      # 生活服务
    ├── 吃饭/                    # 饮食相关
    ├── 舞蹈/                    # 舞蹈教学网站
    ├── 相亲/                    # 相亲交友平台
    └── person/                  # 个人信息展示
```

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/xinshenghu/html5.git
   ```

2. **进入项目目录**
   ```bash
   cd html5
   ```

3. **打开主页面**
   ```bash
   # 使用浏览器打开
   open index.htm  # macOS
   start index.htm # Windows
   ```

## 🛡️ 技术栈
